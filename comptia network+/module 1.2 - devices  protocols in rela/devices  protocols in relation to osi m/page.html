Devices & Protocols in Relation to OSI Model

- MAC Addresses:
  - Layer 2  - data link layer
  - Its a physical address

- IP Addresses:
  - Layer 3 - Network layer
  - Logical addresses
  - Routable

- EUI - 64 (IPv6 Global Unicast)
  - Layer 2 - data link layer
  - IPv6 Global Unicast - unique for single device accross the globe
  - Does not require routing, hence layer 2

- Frames
  - Layer 2 Protocol delivery units
 
- Packets
  - Layer 3 Protocol Delivery Units (PDUs)

- Switches
  - Network devices to connect multiple devices together
  - Layer 2 device as these do not require IP address for communication
  - Performs with help of MAC address mappings

- Routers
  - Allows us to connect to multiple different networks
  - Layer 3 devices
  - Performs routing

- Hub
  - Dumb switch
  - Just sends out the messege recieved to all the other connected nodes

- Encryption Devices
  - Layer 6 devices
  - Could be physical devices or software based

- Cable
  - Layer 1 device

- Network Interface Card
  - Layer 1 device

- Bridge
  - Layer 2 device as it does not do any routing and provides point to point connection
