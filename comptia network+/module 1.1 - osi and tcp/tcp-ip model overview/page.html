TCP/IP Model Overview

- The TCP/IP describes the network structure in 4 Layers
- Layers in TCP/IP consist of:
  - Layer 1 - Network Interface Layer (Layer 1 and 2 of OSI Model)
  - Layer 2 - Internet Layer (Layer 3 of OSI Model)
  - Layer 3 - Transport Layer (Layer 4 of OSI Model)
  - Layer 4 - Application Layer (Layers 5, 6, and 7 of OSI Model)
