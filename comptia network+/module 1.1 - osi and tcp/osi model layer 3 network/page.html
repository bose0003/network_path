OSI Model Layer 3 Network

- Layer 3
- Network Layer
- Commonly reffered to as IP layer
- Layer 3 allows us to route data through multiple networks.
- Fundamental means of transferring data through one network to another.
- Translates logical addresses to physical addresses
- Performs:
  - Network Routing Functions
  - Fragmentations / reassembly
- Routers are layer 3 devices.
